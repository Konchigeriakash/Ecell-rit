# Technical Stack - AI for Internships Platform

### Core Framework & Language
- **Framework**: Next.js 14 (with App Router)
- **Language**: TypeScript

### Styling & UI
- **UI Components**: ShadCN UI
- **CSS Framework**: Tailwind CSS
- **Charts**: Recharts
- **Icons**: Lucide React

### Artificial Intelligence
- **AI Orchestration**: Google's Genkit
- **AI Models**: Google's Gemini family

### Forms & State
- **Form Handling**: React Hook Form
- **Schema Validation**: Zod
- **Client-Side State**: React Hooks with `localStorage` for rapid, persistent prototyping.
